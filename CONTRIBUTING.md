# Contributing to Vibeocracy

Thanks for helping build the open-source home for **Intelligence Over Hierarchy**.

## Quick start

1. Fork [github.com/iCoderBilal/vibeocracy](https://github.com/iCoderBilal/vibeocracy)
2. Clone your fork locally
3. Run a local server (see [README.md](README.md))
4. Make your changes in `index.html` or `assets/`
5. Open a Pull Request against `main`

## What we're looking for

- Copy and messaging improvements
- Performance and accessibility fixes
- Mobile and responsive polish
- Animation and interaction refinements
- SEO and social meta improvements
- Bug fixes

## Guidelines

- **Keep it simple** — this is a static single-page site; avoid adding a build pipeline unless there's a strong reason.
- **Match the aesthetic** — dark `#050505` background, Space Grotesk headings, cyan→pink gradients, glass panels.
- **Test locally** — verify desktop and mobile before submitting.
- **Respect reduced motion** — the site already checks `prefers-reduced-motion`; preserve that behavior.
- **One concern per PR** — smaller, focused PRs are easier to review and merge.

## Pull request checklist

- [ ] Changes work in Chrome, Firefox, and Safari
- [ ] Mobile layout looks correct
- [ ] No console errors on load
- [ ] PR description explains what changed and why

## Questions?

Open a [GitHub Issue](https://github.com/iCoderBilal/vibeocracy/issues) or reach out in the community channel where the project was shared.
